## * SELECT
### - 모든 데이터 출력 : SELECT * FROM 테이블명;
### - 필요한 컬럼만 출력 : SELECT 컬럼명 FROM 테이블명;
### - 필요한 데이터만 출력 : SELECT * FROM 테이블명 WHERE 조건;
### - 정렬 : SELECT * FROM 테이블명 ORDER BY 컬럼명 DESC(or ASC);
### - 레코드 개수 한정 : SELECT * FROM 테이블명 LIMIT 갯수;
